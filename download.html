This program is provided only as source form and also a amd64 debian package.
It is written in plain C (not C++), using GTK 2.0 for the user-interface.
It should compile w/o problem on any Linux Distribution and FreeBSD, as long you have GTK 2.0 development installed.

To download the source, use svn:
git clone https://github.com/osingla/atilo.git

To compile the application, just follow these steps:
cd atilo/Release
make

Please note that atilo is based on GTK2, so the development version of GTK2 needs to be installed on your system.
Typically, on a Debian 9 (stretch), you will need to (at least) install these packages:
apt-get install pkg-config
apt-get install gtk2.0 build-essential libgtk2.0-dev



You can download the amd64 Debian 9 package here:
atilo_1.0_amd64.deb
